# Biblioteca-algoritmo
Projeto desenvolvido para fins de estudo em Análise e Desenvolvimento de Sistemas (ADS).

# Objetivo
O objetivo deste projeto é simular uma biblioteca simples, permitindo armazenar e buscar livros através de um número de identificação (ID).

# Como funciona
Cada livro possui:

* ID (único)
* Título
* Autor

Os livros são armazenados em um dicionário (HashMap), onde o ID é utilizado como chave.
Isso permite que a busca seja feita de forma rápida e eficiente.

# Funcionalidades
* Adicionar livros
* Buscar livro pelo ID

# Tecnologia utilizada
Python

# Estrutura de dados
Foi utilizado um dicionário (HashMap), pois permite buscar os livros de forma rápida.

# Como executar
1. Abra o arquivo `biblioteca.py`
2. Execute o código em um ambiente Python
3. O programa irá mostrar o resultado da busca de um livro
