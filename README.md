# Biblioteca-algoritmo
Projeto desenvolvido para fins de estudo em Análise e Desenvolvimento de Sistemas (ADS).

# Objetivo
O objetivo deste projeto é armazenar e buscar informações de livros utilizando uma estrutura de dados eficiente.

# Como funciona
Cada livro possui:

* ID (único)
* Título
* Autor
Os livros são armazenados em um dicionário, onde o ID é usado como chave para facilitar a busca.

# Funcionalidades
* Adicionar livros
* Buscar livro pelo ID

# Tecnologia utilizada
Python

# Estrutura de dados
Foi utilizado um dicionário (HashMap), pois permite buscar os livros de forma rápida.

# Como executar
1. Abra o arquivo `biblioteca.py`
2. Execute no Python
3. O programa irá mostrar o resultado da busca de um livro
